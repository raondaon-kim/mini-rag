# 사용자 의도 추적 규칙

대화에서 사용자의 목표와 의도를 감지하여 `save_user_intent` 도구로 저장합니다.

## 감지 트리거

다음과 같은 표현이 나타나면 의도를 저장하세요:

| 패턴 | 예시 | 상태 |
|------|------|------|
| 새 목표 | "~하려고 해", "~을 만들 거야" | new |
| 진행 중 | "~하고 있어", "~까지 했어" | in_progress |
| 완료 | "~을 끝냈어", "~이 완성됐어" | done |
| 변경 | "~로 바꿀 거야", "~대신 ~할래" | new (기존 의도 대체) |

## 저장 형식

```
save_user_intent({
  title: "의도 제목 (간결하게)",
  content: "상세 설명 (마크다운, 관련 파일/기술 포함)",
  tags: ["관련", "태그"],
  status: "new" | "in_progress" | "done"
})
```

## 세션 시작 시

새 대화가 시작되면 `get_user_intents`로 이전 의도를 확인하여:
- 진행 중인 의도가 있으면 맥락으로 활용
- 완료된 의도는 참고만

## 주의사항
- 사소한 질문 ("날씨 알려줘")은 의도로 저장하지 마세요
- 프로젝트/작업 관련 목표만 저장하세요
- 같은 의도를 중복 저장하지 마세요
