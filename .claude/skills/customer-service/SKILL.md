# 고객 서비스

## FAQ 생성

```
## 자주 묻는 질문

### [카테고리 1: 예) 주문/배송]

**Q: [고객이 실제로 물어볼 표현 그대로]**
A: [1-3문장으로 명확히. 추가 정보는 링크로]

**Q: [...]**
A: [...]

### [카테고리 2: 예) 결제/환불]
...
```

**FAQ 작성 규칙**:
- 질문은 고객의 언어로 (전문 용어 X)
- 답변은 3문장 이내 (길면 별도 가이드 링크)
- 가장 빈번한 질문을 맨 위에
- 부정형보다 긍정형 ("불가합니다" → "이렇게 하시면 됩니다")

## 응답 템플릿

### 문의 접수
```
[고객명]님, 안녕하세요.
문의해주셔서 감사합니다.

[문의 내용 요약]에 대해 확인해드리겠습니다.
[예상 처리 시간] 이내에 답변드리겠습니다.

추가 문의사항이 있으시면 말씀해주세요.
```

### 문제 해결
```
[고객명]님, 안녕하세요.
[문제]에 대해 불편을 드려 죄송합니다.

확인 결과, [원인]으로 인해 발생한 것으로 파악되었습니다.
[해결 방법/조치 사항]을 진행하였습니다.

[보상/추가 조치가 있다면]
다시 한번 불편을 드린 점 사과드립니다.
```

### 에스컬레이션
```
[고객명]님의 요청 사항을 상위 담당자에게 전달하였습니다.
[예상 응답 시간] 이내에 연락드리겠습니다.
```

## 프로액티브 트리거
- FAQ가 자사 관점으로 작성됨 (고객 질문 형식이어야 함)
- 응답 톤이 로봇적임
- 에스컬레이션 기준이 없음
- 감정적 고객 대응 시나리오가 없음
- 자주 묻는 질문 Top 10이 정리되지 않음
