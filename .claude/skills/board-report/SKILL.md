# 이사회/투자자 보고 가이드

## Board Deck 구조 (10-15 슬라이드)

### 표준 구성
1. **표지** — 회사명, 기간, 기밀 표시
2. **경영 하이라이트** — 3-5개 핵심 성과/이슈
3. **재무 요약** — 매출, 영업이익, 현금흐름 (전년比)
4. **사업부별 실적** — 부문별 매출/이익 테이블
5. **주요 KPI 대시보드** — 신호등 체계 (Green/Yellow/Red)
6. **시장 동향** — 경쟁 환경, 규제 변화
7. **전략 이니셔티브 진행** — 로드맵 대비 진척
8. **리스크 매트릭스** — 영향도 × 발생확률
9. **인력 현황** — 채용, 이직률, 조직 변화
10. **다음 분기 계획** — 목표, 예산, 주요 일정
11. **의결 안건** — 승인 필요 사항
12. **부록** — 상세 데이터

## 투자자 보고 (IR) 추가 요소
- **Unit Economics**: CAC, LTV, LTV/CAC 비율
- **성장 지표**: MRR/ARR, 고객 수, 시장 점유율
- **번레이트**: 월 소진액, 런웨이 (개월)
- **자금 조달**: 라운드, 밸류에이션

## 작성 규칙
1. 모든 수치에 출처/기준일 명시
2. 전년/전기 비교 필수
3. 경영진 코멘트는 팩트 기반
4. 기밀 정보 취급 주의 표시

## 프로액티브 트리거
- 데이터에 기준일이 없음
- 전년/전기 비교가 누락되어 있음
- 리스크를 숨기거나 축소하고 있음
- 의결 안건이 불명확함
- 부록에 핵심 데이터가 묻혀 있음
