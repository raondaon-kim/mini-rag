# 변화관리 가이드

## ADKAR 모델 (Prosci)

### 5단계 개인 변화
| 단계 | 의미 | 핵심 질문 |
|------|------|----------|
| **A**wareness | 변화 필요성 인식 | 왜 바꿔야 하는가? |
| **D**esire | 변화 참여 의지 | 나에게 어떤 이점이? |
| **K**nowledge | 변화 방법 이해 | 어떻게 바꾸는가? |
| **A**bility | 실행 역량 | 할 수 있는가? |
| **R**einforcement | 지속/강화 | 유지되는가? |

### 단계별 활동
| ADKAR | 활동 | 커뮤니케이션 |
|-------|------|------------|
| A | 현황 공유, 위기 인식 | 타운홀, CEO 메시지 |
| D | 비전 제시, WIIFM 설명 | 팀 미팅, 1:1 |
| K | 교육, 매뉴얼, 멘토링 | 교육 세션, FAQ |
| A | 파일럿, 코칭, 피드백 | 실습, Q&A 세션 |
| R | 성과 인정, 보상, 모니터링 | 성공 사례 공유 |

## 이해관계자 분석

### 영향도-관심도 매트릭스
```
        관심도 높음
            │
  적극 관리  │  핵심 플레이어
  (Keep      │  (Manage
   Satisfied)│   Closely)
  ──────────┼──────────
  최소 노력  │  정보 제공
  (Monitor)  │  (Keep
            │   Informed)
            │
        관심도 낮음
  영향도 낮음 ←──→ 영향도 높음
```

## 변화 저항 대응
| 저항 유형 | 원인 | 대응 |
|----------|------|------|
| 정보 부족 | 모르겠음 | 설명, 교육 |
| 불안감 | 내 자리가? | 비전, 역할 명확화 |
| 과거 실패 | 또 실패할 것 | 작은 성공 먼저 |
| 이해관계 충돌 | 내 이익 감소 | WIIFM, 보상 설계 |

## 변화 관리 계획서 구조
1. 변화 정의 (무엇이, 왜)
2. 영향 분석 (누구에게, 얼마나)
3. 이해관계자 전략
4. 커뮤니케이션 계획
5. 교육/지원 계획
6. 저항 관리
7. 성공 지표/모니터링
8. 타임라인

## 프로액티브 트리거
- 변화 이유(Why)가 불명확함
- 영향 분석 없이 시행하고 있음
- 이해관계자 맵이 없음
- 저항 대응 계획이 없음
- 성과 측정이 정의되지 않음
