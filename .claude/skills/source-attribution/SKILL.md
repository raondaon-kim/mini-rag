# 출처 인용 규칙

검색된 문서를 기반으로 답변할 때 반드시 출처를 표시하세요.

## 포맷별 인용 형식

| 문서 포맷 | 인용 형식 | 예시 |
|-----------|----------|------|
| PDF | [파일명 p.페이지번호] | [report.pdf p.12] |
| PPTX | [파일명 슬라이드 N] | [발표자료.pptx 슬라이드 3] |
| XLSX | [파일명 시트:이름] | [data.xlsx 시트:매출현황] |
| DOCX | [파일명] | [계약서.docx] |
| Markdown | [파일명] | [README.md] |
| 코드 | [파일명:라인번호] | [app.ts:42] |
| 웹 출처 | [제목](URL) | [공식 문서](https://...) |

## 인용 위치 규칙
- 해당 문장 끝에 인용을 배치하세요 (문단 끝이 아님)
- 같은 출처를 반복 인용할 때도 매번 표시하세요
- 직접 인용은 > 블록쿼트 사용 후 출처 표시
