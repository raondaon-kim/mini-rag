# 경영진 브리핑 가이드

## 원칙: 1-Page Executive Brief

경영진은 시간이 없다. **30초 안에 핵심을 전달**하라.

### 구조 (BLUF + 피라미드)
1. **결론 먼저** (1-2문장): 가장 중요한 메시지
2. **핵심 수치** (3-5개): 전주/전월 대비 변화
3. **주요 이슈** (2-3개): 의사결정 필요 사항
4. **요청 사항**: 승인/자원/방향 결정

### 1-Page 템플릿
```
━━━ [보고 제목] | [날짜] ━━━

■ 핵심 요약
[결론 1-2문장]

■ 주요 지표
  매출: ₩___M (전월比 +_%)
  비용: ₩___M (예산比 _%)
  [지표3]: ___ (목표比 _%)

■ 이슈 & 의사결정
1. [이슈] → [제안 옵션 A/B] → 권고: [A]
2. [이슈] → [제안] → 권고: [B]

■ 다음 주 주요 일정
- [일정 1]
- [일정 2]

■ 요청 사항
- [구체적 승인/자원 요청]
```

## 경영진 보고 규칙
1. **숫자로 말하라** — "좋아졌다" ✗ → "+12% 증가" ✓
2. **비교 기준 명시** — 전주, 전월, 전년, 목표 대비
3. **옵션과 권고** — 문제만 제시 ✗ → 해결안 + 추천 ✓
4. **1페이지** — 상세는 부록(Appendix)으로
5. **시각화** — 가능하면 차트/그래프 1개

## RAG + 신호등 체계
| 상태 | 의미 | 조건 |
|------|------|------|
| 🟢 Green | 정상 | 목표 달성 or 95%+ |
| 🟡 Yellow | 주의 | 목표의 80-95% |
| 🔴 Red | 위험 | 목표의 80% 미만 |

## 프로액티브 트리거
- 보고서가 1페이지를 초과할 때 → 핵심만 1페이지로 압축, 나머지는 부록
- 결론이 문서 뒤에 배치되었을 때 → BLUF(결론 먼저) 구조로 재배치
- 숫자 없이 정성적 서술만 있을 때 → 정량 지표 + 전기 대비 변동 추가
- 옵션/권고 없이 문제만 보고할 때 → 해결안 옵션 A/B + 권고안 추가
- 비교 기준(전주/전월/목표)이 없을 때 → 비교 축 명시
