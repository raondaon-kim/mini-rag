# Narrative Essay — 내러티브 에세이/수필

**출처**: Michel de Montaigne "수상록" (1580, 에세이 형식 발명) + Vivian Gornick "The Situation and the Story" (2001) + 5-paragraph essay 학술 표준

## 유형별 구조

### A. 개인 에세이 / 수필 (Montaigne 스타일)

```
1. 진입점 — 구체적 순간, 관찰, 질문으로 시작 (논제 X)
2. 탐색과 여담 — 개인 경험 ↔ 외부 관찰 ↔ 독서/문화 참조 오가며
3. 심화/전환 — 표면 아래로, 새로운 깨달음이나 모순 발견
4. 착지 — 해결이 아닌 울림. 오프닝으로 돌아오거나 새로운 이해에 도달
```

**핵심**: 상황(situation) + 이야기(story)가 모두 있어야 함. 
"무슨 일이 있었는가"(상황)만으로는 부족 — "그것이 무엇을 의미하는가"(이야기)가 에세이를 만든다.

### B. 5-Paragraph Essay (학술 표준)

```
1문단: 훅 → 배경 → 논제문(thesis, 마지막 문장)
2문단: 근거 1 — 주제문 → 증거 → 분석 → 전환
3문단: 근거 2 — 주제문 → 증거 → 분석 → 전환
4문단: 근거 3 — 주제문 → 증거 → 분석 → 전환
5문단: 논제 재진술 → 요약 → "So what?" (왜 중요한가)
```

### C. 내러티브 에세이 (자기소개서, 감상문)

```
1. 장면 — 구체적 시간, 장소, 감각 디테일로 시작
2. 촉발 사건 — 갈등, 발견, 변화의 시작
3. 긴장 고조 — 가정이 시험받고, 위기감 형성
4. 전환점 — 새로운 시각을 얻는 순간
5. 성찰 — 변화의 의미, 개인을 넘어 보편적 진실로 연결
```

## 에세이 공통 원칙

- **추상 → 구체**: "행복했다" → "그날 처음으로 뒷골목 고양이에게 참치캔을 열어줬다"
- **"나"가 중심**: 에세이는 글쓴이의 관점과 사고 과정이 핵심
- **발견 과정**: 글쓴이가 쓰면서 무언가를 발견해야 함 (결론을 미리 알고 시작 X)
- **울림으로 끝맺기**: 해결/결론이 아닌, 독자가 계속 생각하게 하는 여운

## 품질 체크
- [ ] 구체적 장면/이미지로 시작하는가? (추상 X)
- [ ] 글쓴이의 사고 과정이 보이는가?
- [ ] 자기 모순이나 불확실성이 포함되었는가? (솔직함)
- [ ] 개인 경험이 보편적 의미로 확장되는가?
- [ ] 독자에게 여운을 남기는 끝맺음인가?

## 프로액티브 트리거
- 도입이 설명/논제로 시작할 때 → 구체적 장면/감각 묘사로 시작하도록 수정
- 갈등/긴장 없이 평면적으로 전개될 때 → 촉발 사건이나 내적 갈등 삽입
- 감각 묘사(시각/청각/촉각/후각/미각)가 부족할 때 → 최소 3가지 감각 디테일 추가
- 결말이 교훈 나열로 끝날 때 → 여운과 울림이 남는 결말로 전환
- 시점(1인칭/3인칭)이 불일치할 때 → 일관된 시점으로 통일
