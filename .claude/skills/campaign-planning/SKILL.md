# 마케팅 캠페인 기획 가이드

## 캠페인 기획 프레임워크

### 1. 캠페인 브리프
| 항목 | 내용 |
|------|------|
| 캠페인명 | |
| 목적 | 인지도/리드/매출/리텐션 |
| 타깃 | 페르소나 + 세그먼트 |
| 핵심 메시지 | 1문장 가치 제안 |
| 기간 | 시작~종료 |
| 예산 | 총액 + 채널별 배분 |
| 성공 KPI | 구체적 수치 목표 |

### 2. 퍼널 설계
```
[노출] → [클릭] → [랜딩] → [전환] → [재구매]
  광고     CTA     LP/양식   구매/가입  리텐션
  
각 단계별:
- 예상 전환율
- 필요 콘텐츠/에셋
- 도구/플랫폼
```

### 3. 채널 믹스
| 채널 | 역할 | 예산% | KPI |
|------|------|-------|-----|
| 검색 광고 | 수요 포착 | | CPC, ROAS |
| SNS 광고 | 인지도 | | CPM, 도달 |
| 이메일 | 너처링 | | 오픈율, CTR |
| 콘텐츠 | SEO/신뢰 | | 트래픽, 체류 |

### 4. 타임라인
| 단계 | 기간 | 활동 |
|------|------|------|
| 사전 준비 | 2주 | 에셋 제작, 랜딩페이지 |
| 런칭 | 1주 | 광고 집행, PR |
| 최적화 | 2-4주 | A/B 테스트, 예산 재배분 |
| 정리 | 1주 | 성과 보고, 인사이트 |

### 5. A/B 테스트 계획
- **테스트 대상**: 헤드라인, CTA, 이미지, 오퍼
- **표본 크기**: 통계적 유의미성 확보 기준
- **기간**: 최소 1-2주
- **판단 기준**: 전환율, CTR, 수익

## 산출물 체크리스트
- [ ] 캠페인 브리프 완성
- [ ] 퍼널 각 단계 에셋 준비
- [ ] 예산 배분 확정
- [ ] 타임라인 수립
- [ ] 성과 측정 대시보드 준비

## 프로액티브 트리거
- 목표 KPI가 구체적 수치가 아닐 때 → "인지도 향상" → "노출 100만회, CTR 2%" 등 수치화
- 예산 배분이 없을 때 → 채널별 예산 배분 비율 + 총액 명시
- 타임라인이 없을 때 → 사전 준비/런칭/최적화/정리 단계별 일정 수립
- A/B 테스트 계획이 없을 때 → 최소 1개 테스트 요소(헤드라인/CTA 등) 설계
- 성과 측정 대시보드가 미준비일 때 → 퍼널 단계별 KPI 추적 대시보드 설계
