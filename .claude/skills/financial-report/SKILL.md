# 재무 보고서

## 월간/분기 재무 보고서 구조

```
1. 요약 (BLUF)
   - 핵심 KPI: 매출, 영업이익, 현금흐름
   - 전기 대비 변동 (▲▼ + %)

2. 매출 분석
   - 제품/서비스별 매출
   - 전년 동기 대비, 전월 대비
   - 주요 증감 원인

3. 비용 분석
   - 고정비 vs 변동비
   - 비용 항목별 상세
   - 예산 대비 실적 (Over/Under)

4. 손익 요약
   | 항목 | 당월 | 전월 | 증감 | 예산 | 차이 |
   |------|------|------|------|------|------|
   | 매출 | | | | | |
   | 매출원가 | | | | | |
   | 매출총이익 | | | | | |
   | 판관비 | | | | | |
   | 영업이익 | | | | | |

5. 현금흐름
6. 주요 리스크 및 전망
7. 권고 사항
```

## 예산 템플릿

```
| 항목 | 1월 | 2월 | ... | 12월 | 연간 합계 |
|------|-----|-----|-----|------|----------|
| **매출** | | | | | |
| 제품 A | | | | | |
| 제품 B | | | | | |
| **비용** | | | | | |
| 인건비 | | | | | |
| 임차료 | | | | | |
| 마케팅 | | | | | |
| **순이익** | | | | | |
```

## 엑셀 재무 규칙 (IBCS 표준)
- **파란 글씨** = 직접 입력 (하드코딩)
- **검정 글씨** = 수식 계산
- **모든 가정은 별도 셀** (Assumptions 시트)
- **숫자를 Python으로 계산 후 하드코딩 금지** — 반드시 Excel 수식 사용
- 단위: 헤더에 명시 (백만원, %)
- 음수: 괄호 표기 (1,234)

## 프로액티브 트리거
- 절대값만 제시하고 증감률이 없음
- 비교 기준(전기, 예산, 전년 동기)이 없음
- 이상치에 대한 설명이 누락되어 있음
- 단위가 불일치하거나 명시되지 않음
- 예산 대비 실적 비교가 없음
