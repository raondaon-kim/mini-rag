# 교재/도서 기획 가이드

## 도서 기획서 구조

### 1. 기획 개요
| 항목 | 내용 |
|------|------|
| 도서명 (가제) | |
| 저자/집필진 | |
| 대상 독자 | 수준, 직무, 학년 |
| 도서 유형 | 교재/참고서/입문서/실습서/워크북 |
| 판형/분량 | A4/B5/신국판, 예상 페이지 |
| 출간 목표일 | |
| 경쟁 도서 | 제목, 차별점 |

### 2. 기획 의도
- **왜 이 책이 필요한가?** (시장 Gap)
- **기존 도서와의 차별점** (최소 3가지)
- **독자가 얻는 가치** (읽기 전 vs 후)

### 3. 목차 구성 원칙
```
Part → Chapter → Section → Subsection

Part I. 기초 (30%)
  Ch.1 개요
    1.1 정의와 배경
    1.2 핵심 개념
  Ch.2 기본 원리
Part II. 핵심 (40%)
  Ch.3-6 주요 내용
Part III. 심화/응용 (20%)
  Ch.7-8 고급 주제
Part IV. 부록 (10%)
  용어집, 참고문헌, 색인
```

**구성 비율 가이드**:
- 입문서: 기초 40% / 핵심 40% / 심화 10% / 부록 10%
- 교재: 기초 20% / 핵심 50% / 심화 20% / 부록 10%
- 전문서: 기초 10% / 핵심 40% / 심화 40% / 부록 10%

### 4. 챕터 구조 템플릿
| 요소 | 역할 | 분량 |
|------|------|------|
| 도입 질문 | 호기심 유발 | 1/2 페이지 |
| 학습목표 | 이 장의 목표 | 3-5줄 |
| 본문 | 개념 + 예시 + 그림 | 15-25 페이지 |
| 박스/칼럼 | 심화, 사례, TIP | 2-3개/장 |
| 요약 | 핵심 3-5줄 | 1/2 페이지 |
| 연습문제 | 복습 + 응용 | 5-10문항 |

## 원고 집필 가이드

### 문체 기준
| 항목 | 교재 | 대중서 |
|------|------|--------|
| 문체 | ~입니다/~습니다 | ~해요/~하죠 |
| 주어 | 생략 가능 | 저자 주어 활용 |
| 용어 | 초출 시 정의+원어 병기 | 쉬운 표현 우선 |
| 그림 | 챕터당 3-5개 | 자유 |

### 분량 산정
| 판형 | 한 페이지 글자 수 | 200페이지 분량 |
|------|-----------------|-------------|
| A4 | 800-1,000자 | 160,000-200,000자 |
| B5 | 600-800자 | 120,000-160,000자 |
| 신국판 | 500-700자 | 100,000-140,000자 |

## 프로액티브 트리거
- 목차가 나열식 (깊이 없음) → Part/Chapter 계층 구조화
- 경쟁 도서 분석 없음 → 차별점 도출 필수
- 연습문제/실습 없는 교재 → 각 장에 문제 추가
- 분량 계획 없음 → 판형별 페이지 산정
- 집필 일정 없음 → 챕터별 마감 일정 수립
