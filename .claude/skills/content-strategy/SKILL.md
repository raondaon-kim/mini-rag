# 콘텐츠 전략 가이드

## 프레임워크: Content Strategy Canvas

### 1. 목표 정의
- **비즈니스 목표**: 브랜드 인지도 / 리드 생성 / 고객 유지 / 매출 증대
- **KPI**: 트래픽, 전환율, 참여율, 공유 수, 리드 수

### 2. 오디언스 분석
| 항목 | 내용 |
|------|------|
| 타깃 페르소나 | 역할, 고민, 정보 소비 습관 |
| 구매 여정 단계 | 인지 → 고려 → 결정 → 유지 |
| 선호 채널 | 블로그, 뉴스레터, SNS, 영상 |

### 3. 콘텐츠 유형 매트릭스
| 여정 단계 | 콘텐츠 유형 | 목적 |
|-----------|------------|------|
| 인지 | 블로그, 인포그래픽, SNS | 트래픽, 브랜드 |
| 고려 | 백서, 케이스스터디, 웨비나 | 리드 확보 |
| 결정 | 비교표, 데모, 고객 후기 | 전환 |
| 유지 | 뉴스레터, 가이드, 커뮤니티 | 리텐션 |

### 4. 콘텐츠 캘린더 구조
```
| 주차 | 테마 | 채널 | 유형 | 담당자 | 상태 |
|------|------|------|------|--------|------|
| W1   |      |      |      |        | 기획 |
| W2   |      |      |      |        |      |
```

### 5. 배포 전략
- **자체 채널**: 블로그, 이메일, 앱
- **유료 채널**: 검색 광고, SNS 광고, 디스플레이
- **획득 채널**: SEO, PR, 게스트 포스팅, 인플루언서

## 품질 체크
- [ ] 목표와 KPI가 측정 가능한가?
- [ ] 각 페르소나별 콘텐츠가 매핑되었는가?
- [ ] 캘린더에 빈 주차가 없는가?
- [ ] 채널별 톤앤매너가 정의되었는가?

## 프로액티브 트리거
- KPI가 측정 불가능할 때 → 구체적 수치 목표로 재정의 (예: "인지도 향상" → "월 유기 트래픽 10,000")
- 페르소나 없이 콘텐츠를 기획할 때 → 최소 1개 타깃 페르소나 정의 후 기획
- 채널별 전략이 미분화(모든 채널에 동일 콘텐츠)일 때 → 채널별 톤/포맷/빈도 차별화
- 캘린더에 빈 주차가 있을 때 → 콘텐츠 갭 식별 및 재활용/큐레이션 콘텐츠로 보충
- 경쟁사 콘텐츠 분석이 없을 때 → 주요 경쟁사 3개사 콘텐츠 벤치마킹 추가
