# 전략 기획

## 사업계획서 구조

```
1. 요약 (Executive Summary) — 전체의 10%, 마지막에 작성
2. 회사 개요 — 미션, 비전, 핵심 가치
3. 시장 분석 — 시장 규모(TAM/SAM/SOM), 트렌드, 고객 세그먼트
4. 제품/서비스 — 가치 제안, 차별점, 경쟁 우위
5. 마케팅 전략 — 4P(Product/Price/Place/Promotion)
6. 운영 계획 — 조직, 프로세스, 기술 인프라
7. 재무 계획 — 3년 추정(매출, 비용, 손익), 손익분기점
8. 리스크 분석 — 주요 리스크 + 완화 방안
9. 실행 로드맵 — 분기별 마일스톤
10. 부록 — 상세 재무, 시장 데이터
```

## OKR 설정

```
## Objective: [정성적 목표 — 영감을 주는 문장]

### Key Result 1: [정량적 측정 가능]
- 현재: [baseline]
- 목표: [target]
- 기간: [Q1 2026]

### Key Result 2: [정량적]
- 현재: → 목표:

### Key Result 3: [정량적]
- 현재: → 목표:
```

**OKR 규칙**:
- Objective는 정성적, 영감적 (숫자 없음)
- Key Result는 반드시 숫자로 측정 가능
- 3-5개 KR/Objective
- 70% 달성이면 성공 (야심적으로 설정)

## 시나리오 플래닝

```
| 시나리오 | 확률 | 전제 조건 | 영향 | 대응 전략 |
|---------|------|----------|------|----------|
| 최선 (Bull) | 20% | [조건] | [긍정 영향] | [공격적 투자] |
| 기본 (Base) | 60% | [조건] | [예상 영향] | [계획대로] |
| 최악 (Bear) | 20% | [조건] | [부정 영향] | [비용 절감] |
```

## 프로액티브 트리거
- 비전/미션 없이 전술만 나열하고 있음
- OKR의 Key Result가 측정 불가능함
- 실행 일정이나 담당자가 지정되지 않음
- 리스크 분석이 누락되어 있음
- 현황 분석 없이 바로 전략을 수립하고 있음
