# 영업/세일즈 문서

## 영업 이메일 시퀀스

### 1차 (콜드 아웃리치)
```
제목: [상대 회사/개인의 관심사 언급]
본문:
- 훅: [최근 뉴스/성과 언급 — 리서치 증명]
- 가치: "저희가 [유사 기업]에서 [구체적 성과]를 달성했습니다"
- CTA: "15분 통화로 [구체적 가치]를 공유하고 싶습니다"
```

### 2차 (후속)
```
"지난 메일 확인하셨는지 궁금합니다"
+ 새로운 가치 제안 1개 추가
+ 더 쉬운 CTA ("자료만 보내드릴까요?")
```

### 3차 (최종)
```
"마지막으로 연락드립니다"
+ 간결하게 핵심 가치 1줄
+ "관심 있으시면 언제든 연락주세요"
```

## 가격 제안서

```
1. 고객 현황 및 니즈 요약
2. 제안 솔루션 개요
3. 가격표
   | 항목 | 수량 | 단가 | 금액 |
   |------|------|------|------|
   | ... | ... | ... | ... |
   | **합계** | | | **₩XX** |
4. 할인/특별 조건 (있을 시)
5. 결제 조건
6. 유효 기간
7. 다음 단계
```

## 경쟁사 비교표

```
| 항목 | 자사 | 경쟁사 A | 경쟁사 B |
|------|------|---------|---------|
| 가격 | ₩X | ₩Y | ₩Z |
| 핵심 기능 1 | ✅ | ✅ | ❌ |
| 핵심 기능 2 | ✅ | ❌ | ✅ |
| 고객 지원 | 24/7 | 평일만 | 이메일만 |
| **차별점** | [핵심 강점] | | |
```

## 프로액티브 트리거
- 콜드메일이 자사 소개로 시작함 (고객 문제로 시작해야 함)
- 본문이 150자를 초과함
- 개인화가 없음
- CTA가 30분 미팅 요청임 (부담 낮은 것부터 시작해야 함)
- 팔로업이 단순 "확인 차" 메일임
