# Creative Presentation — Duarte Sparkline + Hero's Journey

## Duarte Sparkline (Nancy Duarte, "Resonate" 2010)

**적용**: 키노트, 제품 런칭, 변화 관리, 영감을 주는 발표

"현재(What Is)"와 "가능한 미래(What Could Be)" 사이를 오가며 긴장과 희망을 반복합니다.

### 구조

```
슬라이드 1-2: 현재 상태 (공감대)
  → "우리 모두 알고 있듯이..."

슬라이드 3-4: 가능한 미래 (비전 첫 등장)
  → "하지만 만약 이렇게 된다면?"

슬라이드 5-6: 다시 현실 (장애물)
  → "현실적으로 이런 문제가 있습니다"

슬라이드 7-8: 해결의 실마리 (증거)
  → "실제로 이것이 가능하다는 증거가 있습니다"

슬라이드 9-10: 새로운 세계 (비전 완성)
  → "이렇게 변화된 모습을 상상해보세요"

슬라이드 11-12: 행동 촉구 (Call to Adventure)
  → "지금 바로 시작합시다"
```

## Hero's Journey (Joseph Campbell 기반)

**적용**: 제품 스토리, 고객 사례, 브랜드 내러티브

### 12장 템플릿

| # | 슬라이드 | 내용 |
|---|---------|------|
| 1 | 훅 | 놀라운 통계 또는 질문 |
| 2 | 주인공 소개 | "이것은 [고객/사용자]의 이야기입니다" |
| 3 | 일상 세계 | 주인공의 현재 상태 |
| 4 | 문제 등장 | 겪고 있는 고통 |
| 5 | 기존 해결책 실패 | 왜 기존 방법이 안 되는지 |
| 6 | 새 솔루션 등장 | 우리 제품/서비스 소개 |
| 7 | 작동 원리 | 어떻게 문제를 해결하는지 |
| 8 | 증거 1 | 데이터, 차트 |
| 9 | 증거 2 | 고객 사례 / 테스티모니얼 |
| 10 | 변화된 세계 | 솔루션 적용 후의 모습 |
| 11 | 성과 | 결과 수치 |
| 12 | 행동 촉구 | "지금 시작하세요" + 연락처 |

## 크리에이티브 발표 디자인 원칙

- 슬라이드당 **6-15단어** (시각 중심)
- 풀블리드 이미지 + 최소 텍스트 (PechaKucha 스타일)
- 감정적 대비가 핵심 — 현실의 고통 ↔ 미래의 희망
- 발표자 노트에 상세 내러티브 포함

## 프로액티브 트리거
- 모든 슬라이드가 동일 레이아웃 (변화/리듬 필요)
- 이미지/비주얼이 부족하고 텍스트 위주임
- 스토리 아크(현실↔미래 교차)가 없음
- 오프닝 Hook이 없이 바로 내용으로 진입
- 감정적 대비(고통↔희망)가 부족함

## 산출물
- Sparkline/Hero's Journey 구조 PPTX (10-12장)
- 비주얼 스토리보드 (슬라이드별 이미지 방향)
- 발표자 노트 (내러티브 스크립트 전체)
- 품질 체크 완료 리포트
