# 학습 평가 설계 가이드

## 평가 유형 선택

| 유형 | 목적 | 시점 | 예시 |
|------|------|------|------|
| 진단평가 | 사전 수준 파악 | 학습 전 | 사전 테스트 |
| 형성평가 | 학습 과정 점검 | 학습 중 | 퀴즈, 활동, 피드백 |
| 총괄평가 | 학습 성취 판정 | 학습 후 | 기말시험, 프로젝트 |

## 문항 출제 (Bloom's Taxonomy 정렬)

### 객관식 문항 구조
```
[문두]: 명확한 질문 (부정문 사용 자제)
  (1) 정답
  (2) 매력적 오답 (흔한 오개념 기반)
  (3) 매력적 오답
  (4) 매력적 오답
```

**출제 규칙**:
- 문두에 핵심 내용 포함 (선택지에서 반복 X)
- "모두 고르시오" 대신 단일 최선답
- 선택지 길이 균일하게
- "위의 모두" / "해당 없음" 지양

### 수준별 문항 예시
| Bloom 수준 | 문항 유형 |
|-----------|---------|
| 기억 | 사실 확인, 용어 정의 |
| 이해 | 사례 판별, 재진술 |
| 적용 | 새로운 상황에 원리 적용 |
| 분석 | 데이터 해석, 원인 추론 |
| 평가 | 최선 방안 판단, 비교 평가 |
| 창조 | 설계, 기획, 산출물 제작 |

## 루브릭 설계

### 분석적 루브릭 템플릿
| 평가 기준 | 우수 (4) | 양호 (3) | 보통 (2) | 미흡 (1) | 배점 |
|----------|---------|---------|---------|---------|------|
| 내용 정확성 | | | | | /20 |
| 논리적 구성 | | | | | /20 |
| 근거 활용 | | | | | /15 |
| 표현/전달 | | | | | /10 |

**루브릭 작성 규칙**:
- 각 수준의 기술이 관찰 가능해야 함
- 수준 간 차이가 명확해야 함
- 긍정적 진술 사용 (부족한 점이 아닌 도달 기준)

## 문항 분석

### 문항 양호도 지표
| 지표 | 공식 | 양호 기준 |
|------|------|----------|
| 난이도 (P) | 정답자 / 전체 | 0.30 ~ 0.70 |
| 변별도 (D) | 상위-하위 정답률 차 | ≥ 0.30 |
| 신뢰도 (KR-20) | 내적 일관성 | ≥ 0.70 |

## 프로액티브 트리거
- 문항이 모두 기억 수준 → 고차 사고력 문항 추가 권고
- 난이도가 0.90 이상 → 너무 쉬움, 변별력 없음
- 루브릭 수준 간 기술이 모호함 → 관찰 가능한 동사로 수정
- 평가와 학습목표 불일치 → 정렬표(Alignment Matrix) 작성
- 수행평가 없이 객관식만 → 다양한 평가 방법 혼합
