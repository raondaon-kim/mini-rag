# 커리큘럼 설계 가이드

## 커리큘럼 설계 프레임워크

### 1. 역량 체계 정의
```
직무/역할 → 핵심 역량 도출 → 역량별 수준 정의 → 교육 매핑
```

| 역량 | Level 1 (입문) | Level 2 (실무) | Level 3 (심화) | Level 4 (전문가) |
|------|-------------|-------------|-------------|--------------|
| | 개념 이해 | 독립 수행 | 응용/교육 가능 | 혁신/설계 |

### 2. 학습 경로 (Learning Path)
```
[사전 진단] → [필수 기초] → [핵심 과정] → [심화 선택] → [실습/프로젝트] → [인증]
                    │              │              │
                    └──── 분기점: 수준별 분리 ────┘
```

### 3. 교과목 체계
| 구분 | 과정명 | 대상 | 시수 | 선수과목 | 역량 연결 |
|------|--------|------|------|---------|----------|
| 필수 | | | h | | |
| 선택 | | | h | | |

### 4. 시수 계획
| 학습 유형 | 시간 비율 | 방법 |
|----------|---------|------|
| 이론/강의 | 30-40% | 강의, 영상, 읽기자료 |
| 실습/활동 | 40-50% | 워크숍, 실습, 토론 |
| 평가/성찰 | 10-20% | 시험, 프로젝트, 회고 |

**시수 산정 기준**:
- 1학점 ≈ 15시간 (대학 기준)
- 기업 교육: 모듈당 2-4시간
- 온라인 과정: 챕터당 15-30분

## 커리큘럼 맵 (시각화)

### 연간 교육 로드맵
```
        Q1          Q2          Q3          Q4
기초   [과정A]
핵심              [과정B]     [과정C]
심화                          [과정D]     [과정E]
인증                                      [최종평가]
```

### 역량-교과목 매트릭스
| | 과정A | 과정B | 과정C | 과정D |
|---|---|---|---|---|
| 역량1 | ● 주 | ○ 부 | | |
| 역량2 | | ● 주 | ● 주 | |
| 역량3 | | | ○ 부 | ● 주 |

(● 주: 주요 다룸, ○ 부: 보조적 다룸)

## 교육 상품화 (비즈니스)
| 상품 유형 | 구성 | 가격 모델 |
|----------|------|----------|
| 단일 과정 | 1개 교과목 | 건당 결제 |
| 번들 패키지 | 3-5개 과정 묶음 | 할인 패키지 |
| 구독 | 전체 과정 접근 | 월/년 구독 |
| 인증 프로그램 | 필수+심화+평가 | 프리미엄 |

## 프로액티브 트리거
- 역량 체계 없이 과목 나열 → 역량 먼저 정의
- 선수과목 관계 순환 → 의존성 그래프 점검
- 이론 비율 60% 초과 → 실습/활동 비율 높이기
- 모든 과목이 같은 수준 → 수준별 분화 필요
- 평가 없는 학습 경로 → 마일스톤 평가 추가
