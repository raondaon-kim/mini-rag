# Three-Act Structure — 3막 구조

**출처**: 아리스토텔레스 "시학" (기원전 335년) → Syd Field "Screenplay" (1979). 2,300년간 검증된 이야기 구조의 기본.

## 구조

### 1막 — Setup (전체의 25%)
```
1. 오프닝 이미지 — 분위기와 세계관 확립
2. 주인공 소개 — 일상 세계에서의 모습
3. 결핍/욕구 확립 — 무엇이 부족한지
4. 사건 발생 (Inciting Incident) — 일상을 깨뜨리는 사건
5. 1막 전환점 — 주인공이 여정을 시작하기로 결심
```

### 2막 — Confrontation (전체의 50%)
```
6. 장애물 연쇄 — 점점 어려워지는 시련들
7. 중간점 (Midpoint) — 큰 반전 또는 새로운 진실 발견
8. 상황 악화 — 장애물 격화, 아군 상실
9. 위기 (All Is Lost) — 최저점, 실패 직전
10. 2막 전환점 — 해결의 열쇠를 발견
```

### 3막 — Resolution (전체의 25%)
```
11. 클라이맥스 — 중심 갈등과의 최종 대결
12. 해결 — 새로운 균형 수립
13. 마감 이미지 — 오프닝과 대비/거울, 변화를 증명
```

## Show Don't Tell 원칙 (Anton Chekhov)

이야기 내 감정 표현에 적용:

| Tell (X) | Show (O) |
|----------|----------|
| 그녀는 슬펐다 | 그녀는 촛농에 손톱 자국을 냈다 |
| 그는 관대했다 | 아무도 안 볼 때 마지막 만 원을 팁 통에 넣었다 |
| 날씨가 나빴다 | 우산살이 뒤집어지며 비가 옆으로 쏟아졌다 |

- 추상적 감정 단어 → 구체적 행동/이미지로 교체
- 5가지 감각 중 최소 3개 활용
- 대화에는 서브텍스트(말하지 않는 것)가 있어야 함

## 품질 체크
- [ ] 1막에서 주인공의 결핍/욕구가 확립되었는가?
- [ ] Inciting Incident가 전체의 10-15% 지점에 있는가?
- [ ] 장애물이 점점 어려워지는가? (에스컬레이션)
- [ ] All Is Lost가 진짜 절망적으로 느껴지는가?
- [ ] 클라이맥스가 외적 갈등 + 내적 성장 모두 해결하는가?
- [ ] 마감 이미지가 오프닝과 대비되는가?

## 프로액티브 트리거
- 1막이 전체의 25%를 초과할 때 → 셋업을 압축하고 Inciting Incident를 앞당김
- 2막에 미드포인트(반전/진실 발견)가 없을 때 → 중간점 이벤트 삽입
- 3막이 급전개(클라이맥스가 너무 빠르거나 해결이 안일)일 때 → 클라이맥스 장면 확장
- 주인공의 변화 아크(내적 성장)가 없을 때 → 1막 결핍과 3막 변화를 연결
- 서브플롯이 메인 플롯과 연결되지 않을 때 → 서브플롯이 메인 갈등을 반영하도록 조정
