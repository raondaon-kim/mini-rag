# 경력 경로 분석 가이드

## 분석 프레임워크

### 1단계: 교육 이력 → 역량 추출

교육/자격/경력 데이터에서 역량을 추출합니다.

| 데이터 유형 | 추출 역량 | 예시 |
|------------|----------|------|
| **학위/전공** | 기초 역량 | 경영학 → 재무분석, 조직관리 |
| **자격증** | 전문 역량 | 신용분석사 → 재무제표 분석, 신용평가 |
| **수료 교육** | 실무 역량 | Python 데이터분석 → 데이터 처리, 시각화 |
| **경력** | 응용 역량 | 대출심사 3년 → 리스크 관리, 고객 상담 |

### 2단계: 역량 수준 평가

```
Level 1 (인식) — 개념을 알고 있음
Level 2 (이해) — 설명할 수 있음
Level 3 (적용) — 독립적으로 수행 가능
Level 4 (분석) — 문제를 진단하고 개선 가능
Level 5 (전문) — 타인을 교육하고 혁신 가능
```

### 3단계: 역량 클러스터링

```
역량 맵
├── 핵심 역량 (Core): 직무 공통
│   ├── 커뮤니케이션
│   ├── 문제 해결
│   └── 디지털 리터러시
├── 직무 역량 (Functional): 특정 직무
│   ├── 재무 분석
│   ├── 데이터 분석
│   └── 프로젝트 관리
└── 리더십 역량 (Leadership): 조직 기여
    ├── 팀 관리
    ├── 전략적 사고
    └── 변화 관리
```

## 경력 경로 매핑

### 수평 이동 (Job Rotation)
현재 역량과 유사한 다른 직무로의 이동:
```
[현재 직무] → [역량 유사도 분석] → [유사 직무 A, B, C]
```

### 수직 이동 (Promotion Path)
```
사원 → 대리 → 과장 → 차장 → 부장
     [필요 역량 Gap]   [필요 역량 Gap]
```

### T자형 인재 경로
```
넓이 (다양한 기초 역량)
─────────────────────
         │
    깊이 (전문 역량)
         │
         │
```

## 출력 형식

### 개인 역량 프로필
```markdown
## [이름] 역량 프로필

### 보유 역량 요약
| 역량 | 수준 | 근거 |
|------|------|------|
| 재무분석 | Level 4 | 신용분석사 자격 + 실무 3년 |
| 데이터분석 | Level 2 | Python 교육 수료 |

### 강점 역량 (Top 3)
1. [역량] — [근거]

### 개발 필요 역량
1. [역량] — [현재 수준 → 목표 수준]

### 추천 경력 경로
1. [경로 A]: [현재] → [다음 직무] → [목표 직무]
```

## RAG 검색 활용
- search_documents로 교육 이력 데이터 검색
- 엑셀 데이터의 헤더(이름, 교육, 자격, 경력)를 기반으로 역량 추출
- 여러 사람의 데이터를 비교 분석 시 테이블 형태로 정리
