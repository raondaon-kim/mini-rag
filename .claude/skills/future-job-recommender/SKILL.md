# 미래 일자리 추천 가이드

## 추천 프로세스

```
[교육 이력/역량 프로필] → [직무 적합도 매칭] → [미래 유망도 가중] → [추천 결과]
```

### 1단계: 직무 적합도 점수 산출

보유 역량과 직무 필요 역량의 매칭률:
```
적합도 = (충족 역량 수 / 필요 역량 수) × 100
```

| 적합도 | 해석 | 추천 등급 |
|--------|------|----------|
| 80%+ | 즉시 전환 가능 | ⭐⭐⭐ 최우선 |
| 60-79% | 단기 교육 후 가능 | ⭐⭐ 추천 |
| 40-59% | 중기 교육 필요 | ⭐ 도전 가능 |
| 40% 미만 | 장기 전환 경로 | 참고 |

### 2단계: 미래 유망도 가중

| 요소 | 가중치 | 판단 기준 |
|------|--------|----------|
| AI 대체 위험도 | 25% | 반복적/규칙 기반 → 높은 위험 |
| 시장 성장률 | 25% | 산업 성장, 채용 수요 증가 |
| 기술 발전 수혜 | 25% | AI/디지털 기술 활용도 |
| 임금 상승 전망 | 25% | 공급 대비 수요, 전문성 |

### 3단계: 최종 추천 점수
```
최종 점수 = 적합도(50%) + 미래 유망도(30%) + 성장 잠재력(20%)
```
