# HR 채용 문서

## 채용공고 (Job Description)

```
## [직무명]

### 회사 소개 (2-3문장)

### 주요 업무
- [핵심 업무 1] (전체 시간의 X%)
- [핵심 업무 2]
- [핵심 업무 3]
(5-7개, 구체적 동사로 시작)

### 자격 요건
**필수**:
- [경력/학력]
- [기술/역량]

**우대**:
- [추가 역량]

### 근무 조건
- 근무 형태: [정규직/계약직/원격]
- 급여: [범위 또는 협의]
- 복리후생: [핵심 3-5개]

### 지원 방법
- 마감일: [날짜]
- 제출 서류: [이력서, 포트폴리오 등]
- 접수: [이메일/채용 사이트]
```

## 면접 질문 설계

### 구조화 면접 (역량 기반)

| 평가 역량 | 질문 (STAR 유도) | 후속 질문 |
|-----------|----------------|----------|
| 문제 해결 | "가장 어려웠던 업무 문제와 해결 과정을 설명해주세요" | "결과는?", "다시 한다면?" |
| 협업 | "팀 갈등 상황에서 어떻게 대처했나요?" | "상대방의 반응은?" |
| 리더십 | "팀을 이끌어 목표를 달성한 경험을 말씀해주세요" | "어려움은?", "배운 점?" |

### 기술 면접
- 직무 관련 실무 시나리오 제시
- "이 상황에서 어떻게 하시겠습니까?" (가설 질문)
- 포트폴리오/과제 리뷰

## 체크리스트
- [ ] JD가 성별/나이 차별 표현 없는가?
- [ ] 필수 자격이 진짜 필수인가? (과도한 요구 제거)
- [ ] 면접 질문이 STAR 구조를 유도하는가?
- [ ] 평가 기준이 사전에 정의되었는가?

## 프로액티브 트리거
- JD에 필수/우대 요건이 구분되어 있지 않음
- 평가 기준이 주관적임
- 면접 질문이 Yes/No형임
- 다양성(Diversity) 고려가 없음
- 급여 범위가 공개되지 않음
