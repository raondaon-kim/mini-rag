# 기업/경쟁사 분석

## SWOT 분석

```
## [기업명] SWOT 분석

| | 긍정적 | 부정적 |
|---|--------|--------|
| **내부** | **Strengths (강점)** | **Weaknesses (약점)** |
| | - [강점 1] | - [약점 1] |
| | - [강점 2] | - [약점 2] |
| **외부** | **Opportunities (기회)** | **Threats (위협)** |
| | - [기회 1] | - [위협 1] |
| | - [기회 2] | - [위협 2] |

### 전략 도출
- SO 전략 (강점으로 기회 활용): [...]
- WO 전략 (약점 보완하여 기회 활용): [...]
- ST 전략 (강점으로 위협 대응): [...]
- WT 전략 (약점+위협 최소화): [...]
```

## 프로액티브 트리거
- 직접 경쟁사만 분석하고 간접/잠재 경쟁사를 누락하고 있음
- 정성적 비교만 나열하고 수치 기반 비교가 없음
- 자사 강점만 부각하고 약점을 은폐하고 있음
- SWOT이 나열식이며 전략(SO/WO/ST/WT)과 연결되지 않음
- 분석 출처 또는 기준일이 표시되지 않음
