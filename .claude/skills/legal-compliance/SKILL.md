# 법무/컴플라이언스 문서

⚠️ **주의**: 이 Skill은 템플릿과 체크리스트를 제공합니다. 법적 구속력이 있는 문서는 반드시 법률 전문가의 검토를 받으세요.

## 계약서 검토 체크리스트

```
## 계약서 검토: [계약명]

### 기본 사항
- [ ] 계약 당사자 정보 정확한가?
- [ ] 계약 기간 명시되었는가?
- [ ] 자동 갱신 조건 확인

### 핵심 조건
- [ ] 대가/보수 조건 명확한가?
- [ ] 지급 조건 (시기, 방법, 지연 시)
- [ ] 업무 범위(SOW) 명확히 정의되었는가?
- [ ] 성과 기준/인수 조건 있는가?

### 리스크 조항
- [ ] 위약금/손해배상 조항 범위 적절한가?
- [ ] 면책 조항 편향되지 않았는가?
- [ ] 지적재산권 귀속 명확한가?
- [ ] 비밀유지(NDA) 범위와 기간
- [ ] 경쟁 금지 조항 범위

### 종료/해지
- [ ] 해지 사유 및 절차
- [ ] 해지 시 정산 방법
- [ ] 존속 조항 (NDA 등)

### 일반 조항
- [ ] 준거법/관할 법원
- [ ] 분쟁 해결 방법 (중재/소송)
- [ ] 불가항력 조항
```

## NDA 템플릿 구조

```
1. 당사자 정보
2. 비밀정보 정의 (범위)
3. 비밀유지 의무
4. 예외 사항 (공개 정보, 독립 개발 등)
5. 유효 기간
6. 위반 시 구제 수단
7. 준거법 및 관할
```

## 개인정보처리방침 핵심 항목

```
1. 수집하는 개인정보 항목
2. 수집 목적
3. 보유 기간
4. 제3자 제공 여부
5. 위탁 처리 현황
6. 이용자 권리 (열람, 정정, 삭제)
7. 개인정보 보호책임자
8. 시행일
```

## 컴플라이언스 감사 체크리스트

```
| 영역 | 점검 항목 | 준수 | 미준수 | 비고 |
|------|----------|------|--------|------|
| 데이터 보호 | 개인정보처리방침 최신인가? | | | |
| 접근 권한 | 최소 권한 원칙 적용? | | | |
| 교육 | 연간 보안 교육 실시? | | | |
| 기록 | 감사 로그 보관? | | | |
```

## 프로액티브 트리거
- 법적 조언으로 오해 가능한 표현이 있음
- 관할 법령이 명시되지 않음
- 개인정보 영향 평가가 없음
- 계약서 핵심 조항(해지, 손배, 관할)이 누락됨
- 최신 법령 미반영 가능성을 고지하지 않음
