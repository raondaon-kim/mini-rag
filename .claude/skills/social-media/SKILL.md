# 소셜미디어 운영 가이드

## 채널별 특성

| 채널 | 주요 포맷 | 톤 | 최적 빈도 |
|------|----------|----|---------:|
| Instagram | 이미지, 릴스, 스토리 | 비주얼, 감성적 | 3-5회/주 |
| LinkedIn | 텍스트, 아티클, 슬라이드 | 전문적, 인사이트 | 2-3회/주 |
| YouTube | 영상, 숏츠 | 교육적, 엔터테인먼트 | 1-2회/주 |
| X/Twitter | 텍스트, 쓰레드 | 간결, 시의성 | 5-7회/주 |
| 블로그 | 장문 콘텐츠 | 깊이, SEO | 1-2회/주 |

## 포스팅 구조

### 텍스트 포스트 (Hook → Story → CTA)
1. **Hook** (첫 줄): 호기심 유발, 숫자, 질문, 반전
2. **Body**: 가치 전달 (팁, 인사이트, 스토리)
3. **CTA**: 댓글 유도, 저장, 공유, 링크

### 캘린더 템플릿
```
| 요일 | 콘텐츠 타입 | 주제 | 채널 | CTA |
|------|-----------|------|------|-----|
| 월   | 교육       |      |      |     |
| 수   | 비하인드    |      |      |     |
| 금   | 사례/후기   |      |      |     |
```

## 콘텐츠 믹스 비율
- **교육/인사이트**: 40% — 전문성 구축
- **엔터테인먼트**: 20% — 참여율 향상
- **프로모션**: 20% — 전환 유도
- **커뮤니티**: 20% — 관계 구축

## 성과 지표
| 지표 | 의미 | 벤치마크 |
|------|------|---------|
| 도달률 | 노출 대비 고유 사용자 | 채널별 상이 |
| 참여율 | (좋아요+댓글+공유)/도달 | 1-3% |
| 클릭률 | 클릭/노출 | 0.5-2% |
| 팔로워 증가율 | 월간 순증 | 2-5%/월 |

## 프로액티브 트리거
- 게시 빈도가 주 2회 미만임
- 100% 홍보성 콘텐츠임
- 참여율 지표가 없음
- 채널별 톤이 분화되지 않음
- 다른 계정과의 상호작용이 없음
