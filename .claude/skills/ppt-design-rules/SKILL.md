# PPT 슬라이드 디자인 원칙 (모든 유형 공통)

## 텍스트 규칙

| 항목 | 규칙 |
|------|------|
| 제목 폰트 | 28-36pt |
| 본문 폰트 | 18-24pt (최소 18pt) |
| 불릿 포인트 | 슬라이드당 최대 4-5개 |
| 단어 수 | 슬라이드당 30-40단어 이내 |
| 6×6 규칙 | 6줄 이하, 줄당 6단어 이하 |

## 빌보드 테스트
> 3초 안에 핵심 메시지를 파악할 수 있는가?

## 구성 원칙

1. **1 슬라이드 = 1 아이디어**
2. **제목 = 결론/주장** (토픽 X) — "Q3 매출" → "Q3 매출 15% 성장"
3. **시각:텍스트 = 60:40** (가능하면)
4. **여백** — 가장자리 10% 이상
5. **일관성** — 폰트, 색상, 정렬 전체 덱 통일
6. **발표자 노트** — 상세 설명은 노트에

## AI PPT 생성 체크리스트

- [ ] 각 슬라이드에 하나의 핵심 메시지?
- [ ] 제목이 결론/주장 형태?
- [ ] 불릿 5개 이하?
- [ ] 텍스트 40단어 이내?
- [ ] 첫 슬라이드에서 "왜 중요한지" 명확?
- [ ] 마지막에 Call to Action?
- [ ] 발표자 노트 포함?

## 프로액티브 트리거
- 폰트가 3종류를 초과함 (2종류 이내 권장)
- 색상이 5개를 초과함 (메인 + 보조 + 강조 3-4개 적정)
- 정렬이 불일치함 (좌정렬/중앙정렬 혼용)
- 여백이 부족함 (가장자리 10% 미만)
- 애니메이션이 과다함 (내용 전달에 방해)
- 빌보드 테스트 실패 (3초 내 핵심 메시지 파악 불가)

## 산출물
- 디자인 규칙 준수 검수 리포트
- 수정된 PPTX (규칙 위반 항목 교정)
- 슬라이드 마스터/템플릿 가이드라인
- 체크리스트 완료 확인서
