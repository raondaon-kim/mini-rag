# 인강/영상 강의 스크립트 가이드

## 강의 구조 (Gagne's 9 Events 기반)

### 1회차 스크립트 구조 (15-20분)
| 단계 | 시간 | 내용 | 화면 |
|------|------|------|------|
| **Hook** | 0:00-0:30 | 흥미 유발 (질문, 사례, 통계) | 타이틀 + 키 비주얼 |
| **목표 제시** | 0:30-1:00 | "이 강의를 마치면 ~할 수 있습니다" | 학습목표 텍스트 |
| **사전 연결** | 1:00-2:00 | 이전 내용 복습, 일상 연결 | 다이어그램 |
| **핵심 내용 1** | 2:00-6:00 | 개념 설명 + 예시 | 슬라이드 + 그래픽 |
| **중간 정리** | 6:00-7:00 | 핵심 요약, 잠깐 멈춤 | 요약 카드 |
| **핵심 내용 2** | 7:00-12:00 | 심화 + 실습/데모 | 화면 공유/실습 |
| **적용** | 12:00-15:00 | 실제 사례 적용, 연습 | 예제 화면 |
| **정리** | 15:00-17:00 | 3줄 요약, 핵심 키워드 | 요약 슬라이드 |
| **다음 예고** | 17:00-18:00 | 다음 차시 미리보기, 과제 | 다음 차시 제목 |

## 스크립트 작성 규칙

### 구어체 전환
| 문어체 (❌) | 구어체 (✓) |
|-----------|----------|
| ~라고 할 수 있다 | ~라고 볼 수 있죠 |
| 이에 대해 살펴보겠다 | 이거 한번 같이 볼까요? |
| 그러므로 | 그래서요 |
| ~것이다 | ~거예요 / ~겁니다 |

### 톤 & 페이스
- **1문장 = 1개념** (복문 분리)
- **호흡 표시**: / (짧은 쉼), // (긴 쉼)
- **강조 표시**: **굵게** 또는 [강조]
- **분당 발화량**: 150-180자 (한국어)
- **15분 강의 ≈ 2,500-2,700자**

## 스토리보드 템플릿
| # | 시간 | 내레이션 | 화면(슬라이드/영상) | 자막/키워드 | 효과 |
|---|------|---------|-------------------|-----------|------|
| 1 | 0:00 | | | | |
| 2 | 0:30 | | | | |

## 강의 유형별 구조

| 유형 | 시간 | 특징 |
|------|------|------|
| 개념 강의 | 10-15분 | 이론 중심, 그래픽 |
| 실습 강의 | 15-25분 | 화면 공유, 따라하기 |
| 토론/사례 | 10-20분 | 사례 제시 → 분석 |
| 퀴즈/복습 | 5-10분 | 문제 풀이, 해설 |

## 시리즈 구성
```
[코스 소개 (무료)] → [기초 모듈 3-5회] → [핵심 모듈 5-8회] → [심화 모듈 3-5회] → [최종 프로젝트]
```

## 프로액티브 트리거
- 대본이 문어체 → 구어체로 전환 제안
- 15분 넘는 단일 강의 → 분리 권고
- 화면 설명 없이 내레이션만 → 비주얼 기획 필요
- 예시/사례 없는 이론 → 반드시 1개 이상 사례 추가
- 학습자 참여 요소 없음 → 질문, 퀴즈, 생각할 거리 삽입
