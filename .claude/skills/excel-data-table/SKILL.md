# Excel Data Table — 표 디자인 원칙

**출처**: Edward Tufte "Data-Ink Ratio" + Jon Schwabish "Ten Guidelines for Better Tables" (2020, Journal of Benefit-Cost Analysis) + 엑셀 모범 사례 종합

## 테이블 구조

```
┌──────────────────────────────────────────┐
│ [헤더 행] 볼드, 진한 배경, 흰 글씨, 필터  │
├──────────────────────────────────────────┤
│ [데이터 행 1]                             │
│ [데이터 행 2]                             │
│ [데이터 행 3]                             │
├──────────────────────────────────────────┤
│ [합계 행] 볼드, 상단 이중선 (있을 때)      │
└──────────────────────────────────────────┘
```

## 서식 규칙

| 항목 | 규칙 |
|------|------|
| 헤더 | 볼드, 배경 #4472C4, 글씨 흰색 |
| 정렬 | 텍스트=왼쪽, 숫자=오른쪽, 날짜=가운데 |
| 열 너비 | 내용에 맞춰 자동, 최소 8 최대 50 |
| 숫자 서식 | 천 단위 콤마, 통화 기호, 소수점 통일 |
| 음수 | 빨간색 또는 괄호 (-$100 또는 ($100)) |
| 날짜 | YYYY-MM-DD 또는 YYYY/MM/DD 통일 |
| 빈 셀 | "-" 또는 "N/A" (완전히 비우지 않기) |

## Data-Ink Ratio (Edward Tufte)

> 데이터 잉크 / 전체 잉크 = **최대화**

**적용**:
- 세로선 최소화 (가로선만 얇게)
- 배경색은 헤더와 합계만
- 교차 행 색상(zebra stripe)은 선택적
- 불필요한 테두리, 그림자, 3D 효과 제거

## 시트 이름 규칙
- 명확하고 간결: "월별매출", "고객목록" (O)
- 시트 탭에서 잘리지 않는 길이 (최대 15자)
- 공백 대신 밑줄: "Raw_Data"
- 순서: 요약 → 상세 → 원시데이터

## Schwabish 10 Guidelines (핵심 요약)

1. 헤더를 본문에서 시각적으로 분리하라
2. 세로선을 사용하지 마라 (또는 최소화)
3. 텍스트=왼쪽, 숫자=오른쪽
4. 소수점을 정렬하라
5. 단일 서체를 사용하라
6. 헤더는 Sentence case (ALL CAPS 금지)
7. 관련 행을 여백으로 그룹화하라
8. 강조(볼드, 색상)는 의도적으로, 드물게
9. 열을 5-6개 이내로 제한 (넘으면 분리)
10. 단위는 헤더에, 개별 셀에 넣지 마라

## 품질 체크

- [ ] 헤더 행이 명확하게 구분되는가?
- [ ] 세로선이 최소화되었는가?
- [ ] 숫자 서식이 열 전체에서 통일되는가?
- [ ] 소수점이 정렬되었는가?
- [ ] 빈 셀이 "-"로 표시되었는가?
- [ ] 합계/소계 행이 볼드로 구분되는가?
- [ ] 열 수가 6-8개 이내인가?
- [ ] 단위가 헤더에만 표시되었는가?
- [ ] 불필요한 장식 요소가 없는가?

## 프로액티브 트리거
- 헤더 행이 없거나 불명확할 때 → 볼드 + 배경색 헤더 행 추가
- 병합 셀이 사용되었을 때 → 병합 해제 (피벗/필터 불가 방지)
- 한 열에 혼합 데이터 유형(숫자+텍스트)이 있을 때 → 열 분리 또는 데이터 정규화
- 빈 행/열이 존재할 때 → 빈 셀은 "-" 또는 "N/A"로 채우고 빈 행 제거
- 데이터 유효성 검증이 없을 때 → 드롭다운, 범위 제한 등 유효성 검사 추가
