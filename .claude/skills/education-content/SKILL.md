# 교육 콘텐츠 제작 가이드

## 콘텐츠 유형별 설계

### 1. 교안 (Lesson Plan)
| 항목 | 내용 |
|------|------|
| 과목/모듈 | |
| 차시 | /전체 |
| 학습목표 | (관찰 가능한 동사로) |
| 대상 | 수준, 인원 |
| 시간 | 분 |

**교안 흐름 (5E 모델)**:
| 단계 | 시간% | 활동 | 교수자 역할 |
|------|-------|------|-----------|
| **Engage** (참여) | 10% | 질문, 영상, 사례 | 동기 유발 |
| **Explore** (탐색) | 20% | 조별 활동, 실험 | 관찰, 안내 |
| **Explain** (설명) | 30% | 개념 강의, 시범 | 설명, 피드백 |
| **Elaborate** (확장) | 25% | 응용 과제, 실습 | 코칭 |
| **Evaluate** (평가) | 15% | 퀴즈, 발표, 성찰 | 평가, 정리 |

### 2. 학습 활동 유형
| 활동 | 적합 상황 | 학습 효과 |
|------|----------|----------|
| 강의/설명 | 새 개념 도입 | 지식 전달 |
| 토론 | 관점 다양화 | 비판적 사고 |
| 사례 분석 | 이론 적용 | 분석력 |
| 역할극/시뮬 | 기술 연습 | 실천력 |
| 프로젝트 | 종합 적용 | 창의성+협업 |
| 피어 리뷰 | 상호 학습 | 평가 능력 |
| 플립러닝 | 사전 학습 | 자기주도 |
| 게이미피케이션 | 동기 부여 | 참여도 |

### 3. 워크시트/학습지 구성
```
[학습목표 안내]
[핵심 개념 요약] (빈칸 채우기 또는 그래픽 오거나이저)
[가이드 연습] (예시 → 따라하기)
[독립 연습] (스스로 풀기)
[성찰] "오늘 배운 것 중 가장 중요한 것은?"
```

### 4. 시각 자료 설계
| 자료 유형 | 용도 | 도구 |
|----------|------|------|
| 개념도 | 관계 시각화 | 다이어그램 |
| 순서도 | 프로세스 | 플로우차트 |
| 비교표 | 차이점 정리 | 표/매트릭스 |
| 인포그래픽 | 데이터 요약 | 그래픽 |
| 타임라인 | 시간순 정리 | 연표 |

## 멀티미디어 콘텐츠 기획

### 영상 콘텐츠
| 유형 | 길이 | 적합 |
|------|------|------|
| 마이크로 러닝 | 3-5분 | 단일 개념, 팁 |
| 강의 영상 | 10-20분 | 주제별 심층 |
| 실습 영상 | 15-30분 | 따라하기, 데모 |
| 인터뷰 | 10-15분 | 전문가 인사이트 |

### 인터랙티브 콘텐츠
- **H5P**: 드래그앤드롭, 핫스팟, 분기 시나리오
- **퀴즈**: 즉각 피드백 포함
- **시뮬레이션**: 가상 실습 환경
- **AI 대화**: 시나리오 기반 연습

## 접근성 (Accessibility)
- 영상: 자막 필수
- 이미지: 대체 텍스트
- 색각 이상 고려: 색만으로 정보 구분 X
- 폰트: 본문 14pt 이상
- 명도 대비: 4.5:1 이상

## 프로액티브 트리거
- 강의만 있고 활동 없음 → 능동적 학습 활동 추가
- 텍스트만 있고 시각 자료 없음 → 다이어그램/표 추가
- 접근성 미고려 → 자막, 대체 텍스트 확인
- 평가 없는 콘텐츠 → 형성평가(퀴즈) 삽입
- 학습목표와 활동 불일치 → 정렬 확인
