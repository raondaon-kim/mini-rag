# 스킬 갭 분석 가이드

## 분석 프로세스

### 1단계: 현재 역량 평가 (AS-IS)
교육 이력, 자격증, 경력에서 보유 역량과 수준을 추출합니다.

### 2단계: 목표 직무 역량 정의 (TO-BE)
목표 직무에 필요한 역량과 최소 수준을 정의합니다.

### 3단계: 갭 산출
```
Gap = 목표 수준 - 현재 수준
```

| Gap 크기 | 의미 | 교육 대응 |
|---------|------|----------|
| 0 이하 | 충족 | 유지/심화 |
| 1 | 소폭 부족 | 단기 교육 (1-2주) |
| 2 | 보통 부족 | 중기 교육 (1-3개월) |
| 3 이상 | 대폭 부족 | 장기 교육 (3개월+) + 실습 |

## 갭 분석 출력 형식

```markdown
## [이름] 스킬 갭 분석 — 목표: [직무명]

### 갭 요약
| 역량 | 현재 | 목표 | Gap | 우선순위 |
|------|------|------|-----|---------|
| [역량A] | L2 | L4 | -2 | 🔴 높음 |
| [역량B] | L3 | L4 | -1 | 🟡 보통 |
| [역량C] | L4 | L4 | 0 | 🟢 충족 |

### 교육 로드맵
| 순서 | 교육 과정 | 기간 | 대상 역량 | 기대 효과 |
|------|----------|------|----------|----------|
| 1 | [과정명] | [기간] | [역량] L2→L3 | [설명] |
| 2 | [과정명] | [기간] | [역량] L3→L4 | [설명] |

### 총 예상 교육 기간: [N개월]
### 우선 추천 교육: [과정명]
```

## 교육 설계 연결
갭 분석 결과를 기반으로 교육 과정을 설계할 때:
1. Gap이 큰 역량 → 기초부터 체계적 교육
2. Gap이 작은 역량 → 실습/프로젝트 중심
3. 충족 역량 → 심화/리더십 교육
4. 여러 사람의 공통 Gap → 그룹 교육 설계
