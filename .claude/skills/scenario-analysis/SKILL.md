# 시나리오 분석 가이드

## 시나리오 플래닝 프레임워크

### 1. 핵심 불확실성 식별
| 불확실성 | 영향도 (1-5) | 불확실도 (1-5) | 우선순위 |
|---------|-------------|---------------|---------|
| | | | 영향×불확실 |

→ 상위 2개를 축(Axis)으로 선정

### 2. 2×2 시나리오 매트릭스
```
                불확실성 A (+)
                    │
    시나리오 2       │      시나리오 1
    (낙관적B+비관적A) │      (양쪽 낙관)
                    │
 ───────────────────┼───────────────── 불확실성 B
                    │
    시나리오 3       │      시나리오 4
    (양쪽 비관)      │      (낙관적A+비관적B)
                    │
                불확실성 A (-)
```

## 프로액티브 트리거
- 불확실성이 3개 초과임 (최대 2축으로 제한해야 함)
- 확률 추정 없이 시나리오만 나열하고 있음
- 대응 전략이 시나리오별로 분화되지 않음
- 민감도 분석이 없음
- 트리거 포인트(시나리오 전환 신호)가 정의되지 않음
