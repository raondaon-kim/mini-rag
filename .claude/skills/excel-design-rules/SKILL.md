# 엑셀 디자인 원칙 (모든 유형 공통)

## 워크북 구조

```
1. 첫 시트: 요약/대시보드 (전체 개요)
2. 중간 시트: 상세 데이터 (주제별 분리)
3. 마지막 시트: 원시 데이터 (Raw_, 선택)
```

## 헤더 표준

- 배경: #4472C4 (진한 파랑)
- 글씨: 흰색, 볼드
- 행 높이: 데이터보다 약간 높게
- 행 고정(freeze): 첫 행 고정 필수
- 자동 필터 활성화

## 수치 서식 표준

| 데이터 유형 | 서식 | 예시 |
|-----------|------|------|
| 일반 숫자 | #,##0 | 1,234 |
| 금액 | #,##0원 또는 $#,##0 | 1,234원 |
| 백분율 | 0.0% | 12.3% |
| 날짜 | YYYY-MM-DD | 2026-04-09 |
| 전화번호 | 000-0000-0000 | 010-1234-5678 |

## AI 엑셀 생성 체크리스트

- [ ] 첫 시트가 요약인가?
- [ ] 시트 이름이 15자 이내로 명확한가?
- [ ] 헤더에 필터가 적용되었는가?
- [ ] 텍스트=왼쪽, 숫자=오른쪽 정렬인가?
- [ ] 열 너비가 내용에 맞는가?
- [ ] 빈 셀에 "-" 표시가 있는가?
- [ ] 숫자 서식(콤마, 소수점)이 통일되었는가?

## 프로액티브 트리거
- 셀 병합 사용 → 피벗/필터 불가, 병합 대신 "가운데 맞춤" 사용
- 숫자 서식 불일치 (같은 열에 다른 포맷) → 열 단위로 서식 통일
- 헤더 행 없이 데이터 시작 → 첫 행을 헤더로 설정 + 필터 적용
- 색상만으로 의미 구분 → 텍스트 라벨 병기 (접근성)
- 시트가 1개에 모든 데이터 → 요약 시트 + 데이터 시트 분리
