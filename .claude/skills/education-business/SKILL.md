# 교육 사업 기획 가이드

## 교육 상품 기획 프레임워크

### 1. 시장 분석
| 분석 항목 | 내용 |
|----------|------|
| 대상 시장 규모 | TAM/SAM/SOM |
| 타깃 페르소나 | 직무, 수준, 학습 동기, 예산 |
| 경쟁 과정 | 가격, 수강생 수, 리뷰 점수, 차별점 |
| 트렌드 | 수요 급증 키워드, 기술 트렌드 |

### 2. 교육 상품 설계
| 상품 유형 | 구성 | 가격대 | 마진 |
|----------|------|--------|------|
| 무료 콘텐츠 | 블로그, 유튜브, 체험판 | 0원 | 리드 획득 |
| 단일 과정 | 10-20회차 인강 | 5-30만원 | 60-80% |
| 부트캠프 | 집중 4-12주 | 100-500만원 | 40-60% |
| 구독 | 전체 라이브러리 | 월 1-5만원 | 70-85% |
| 기업 교육 | B2B 맞춤 | 협의 | 50-70% |
| 인증 프로그램 | 과정+평가+자격 | 50-200만원 | 50-65% |

### 3. 퍼널 설계
```
[인지] 무료 콘텐츠, SNS, SEO
  ↓ (전환율 5-10%)
[체험] 무료 강의, 샘플 챕터, 웨비나
  ↓ (전환율 10-20%)
[구매] 단일 과정 or 체험판
  ↓ (업셀 20-30%)
[업그레이드] 번들, 부트캠프, 구독
  ↓ (재구매율 30-50%)
[충성] 커뮤니티, 고급 과정, 멘토링
```

## 교육 사업 KPI

### 핵심 지표
| 지표 | 공식 | 벤치마크 |
|------|------|---------|
| 수강생 획득 비용 (CAC) | 마케팅비 / 신규 수강생 | 5-30만원 |
| 수강생 생애가치 (LTV) | ARPU × 평균 수강 기간 | CAC의 3배+ |
| 과정 완료율 | 완료자 / 등록자 | 온라인 15-30% |
| NPS | 추천자% - 비추천자% | >50 우수 |
| 재구매율 | 2회+ 구매자 / 전체 | >30% 양호 |

### 교육 ROI (기업 교육용)
```
ROI = (교육 효과 금액 - 교육 비용) / 교육 비용 × 100%

교육 효과 금액 =
  생산성 향상 + 이직률 감소 절감 + 오류 감소 절감 + 매출 증가
```

**Phillips ROI 5단계**:
1. 반응: 만족도
2. 학습: 지식/스킬 변화
3. 적용: 현업 적용도
4. 영향: 비즈니스 성과 변화
5. ROI: 금전적 효과/비용

| 수료 직전 | 동기 저하 | 수료 인센티브, 커뮤니티 |

## 프로액티브 트리거
- 완료율 15% 미만 → 콘텐츠 품질/난이도 점검
- CAC > LTV → 가격 인상 또는 마케팅 효율화
- NPS < 30 → 수강생 인터뷰 즉시 실시
- 재구매율 10% 미만 → 후속 과정/업셀 경로 설계
- 기업 교육 ROI 미측정 → Phillips 5단계 적용
